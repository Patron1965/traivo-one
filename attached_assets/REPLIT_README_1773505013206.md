# Traivo Logo – Instruktioner för Replit

## Om filen

| Egenskap | Värde |
|---|---|
| **Filnamn** | `Traivo_logo_transparent.png` |
| **Storlek** | 700 × 170 px |
| **Format** | PNG med alfakanal (RGBA) |
| **Bakgrund** | ✅ **Helt transparent** – ingen vit eller grå box |
| **Filstorlek** | ~49 KB |

Logotypen har äkta transparens med 7 071 helt transparenta pixlar och 6 129 semi-transparenta pixlar (mjuka kanter/antialiasing), vilket ger ett professionellt utseende mot alla bakgrundsfärger.

---

## Ladda upp till Replit – steg för steg

1. Öppna ditt projekt i Replit
2. Klicka på **Files** (📁) i vänstra panelen
3. Skapa en mapp `images/` eller `public/images/` (beroende på ramverk)
4. Klicka på de tre prickarna (**⋮**) → **Upload file**
5. Välj `Traivo_logo_transparent.png` och ladda upp
6. Verifiera att filen syns i filträdet

> **Tips:** I Next.js/Vite-projekt, lägg filen i `public/images/` så att den serveras statiskt.

## Tips

### Responsiv användning

```css
.logo img {
  max-width: 100%;
  height: auto;
  width: clamp(200px, 50vw, 350px); /* Anpassar sig 200–350px */
}
```

### Dark mode

Eftersom bakgrunden är **helt transparent** fungerar logotypen direkt mot mörka bakgrunder. Om logotypens text/grafik är mörk och behöver bättre kontrast:

```css
@media (prefers-color-scheme: dark) {
  .logo img {
    filter: brightness(1.2);
    /* eller: filter: invert(1); för helt inverterad */
  }
}
```

### Bekräftelse av transparens

> ✅ Filen har verifierats programmatiskt: den innehåller en fullständig alfakanal (RGBA) med äkta transparenta pixlar. Det finns **ingen vit eller grå bakgrundsbox** – logotypen flyter fritt mot vilken bakgrundsfärg som helst.
