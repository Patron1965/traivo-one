**Uppgift B: Auto-notis "Tekniker på väg"**

### 2. DASHBOARD INSIGHTS (Punkt 6)
**Uppgift C: "Kräver uppmärksamhet"-widget**
Skapa ny DashboardAlerts-komponent som visar:
- Försenade order (status !== 'completed' && scheduledEnd < now)
- Tekniker utan jobb idag (0 assignade order)
- Dubbelbokning (samma tekniker, överlappande tider)
- Varje alert klickbar → navigera till relevant vy

**Uppgift D: Kapacitetsöversikt**
- Visa % utnyttjande per tekniker för vald dag
- Beräkning: (bokad tid / arbetstid) * 100
- Färgkodning: <60% grön, 60-90% gul, >90% röd
- Visa i befintlig Dashboard-layout

### 3. SMART CHECKLISTA (Punkt 1)
**Uppgift E: AI-förslag för nästa steg**

  - Historik från liknande order (senaste 10)
  - Nuvarande status

- Spara valda steg i order_checklist-tabell

**Uppgift F: Varning om glömda fält**

- Visa modal med lista över saknade fält
- Blockera signering tills alla fält är ifyllda
