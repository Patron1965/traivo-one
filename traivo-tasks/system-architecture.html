Centrala dataflöden
Dataflöden & Arbetsprocesser

📋 Orderflöde

1. Planerare skapar order (manuellt eller via orderkoncept-wizard)
2. AI Auto-Scheduler föreslår tilldelning + Decision Trace
3. Constraint Engine validerar (7 hårda + 1 mjuk regel)
4. Schedule Diff View visar Före/Efter/Delta för KPI:er
5. Planerare accepterar/avvisar per flytt
6. Order tilldelas resurs → push till Traivo Go
7. Fältarbetare utför → statusuppdatering
8. Avvikelser, material, signatur rapporteras
9. Faktura genereras → Fortnox-export
10. Protokoll och rapporter skapas

🤖 AI-planeringsflöde

1. Hämta oschemalagda ordrar + resurser + kluster
2. Beräkna kluster-matchning (postnummer)
3. Heuristisk tilldelning (kapacitet + geografi)
4. OpenAI förbättrar fördelning (effektivitet)
5. Constraint Engine validerar alla tilldelningar
6. Hårda överträdelser → order tas bort
7. Risk Calculator → score per tilldelning
8. Decision Trace byggs (KPI baseline vs proposed)
9. Resultat med moves, violations, riskfactors
10. Loggas till planning_decision_log

📱 Mobilflöde (Traivo Go)

1. Chaufför loggar in med PIN
2. Check-in → arbetspass startar
3. Synka ner dagens ordrar + artiklar
4. GPS-bakgrundsspårning startar (var 30s)
5. Navigera till första order
6. Utför order → status, material, foto, signatur
7. Avvikelse? → rapport med foto + GPS
8. Offline-kö samlar alla händelser
9. Batch-synk vid nätverksåterkoppling
10. Check-out → ruttbetyg → löneunderlag

🔌 IoT → Automatisk order

1. IoT-enhet skickar signal (nivå, temperatur, etc)
2. API validerar API-nyckel
3. Signal sparas + tröskelvärde kontrolleras
4. Tröskel överskriden → arbetsorder skapas automatiskt
5. AI Prediktivt underhåll analyserar historik
6. Prognos: nästa servicedatum + konfidensbetyg

🌐 Kundportalflöde

1. Kund loggar in via token-länk (e-post/SMS)
2. Ser kommande besök + orderhistorik
3. Kartvy visar alla hämtningsställen
4. Kan boka egna besök (självbokning)
5. Realtids-chatt med planerare
6. QR-scan → anonym felanmälan (interimobjekt)

💰 Ekonomiflöde

1. Utförd order med artiklar + material
2. Prisupplösning: objektartikel → prislista → standard
3. Fakturaförhandsgranskning
4. Batch-val och godkännande
5. Export till Fortnox
6. Exporthistorik + statusspårning
